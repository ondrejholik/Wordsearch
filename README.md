# About
Simple wordsearch(osmismerka) generator
  - export to png
  - adapt rows and columns based on text size

# Install 
  ```bash
  git clone github.com/ondrejholik/wordsearch
  cd wordsearch
  go get -u github.com/fogleman/gg
  go build
  ```
# Run
  `./wordsearch ./small.txt`
